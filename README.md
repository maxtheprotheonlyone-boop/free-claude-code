## 🧰 Troubleshooting

### The app does not start

- Run the app again
- Make sure the download finished
- Check that Windows did not block the file
- Reinstall from the latest release

### The window opens and closes fast

- Open the app from the folder where it was installed
- Keep the command window open if the app uses one
- Try the latest release file again

### The model does not load

- Check your internet connection
- Make sure your PC has enough free space
- Try a smaller model if your GPU memory is low
- Restart the app and try again

### The app feels slow

- Close other large apps
- Use a smaller model
- Make sure NVIDIA drivers are current
- Give the app more free memory by closing unused programs

## 🧩 Tips for better results

- Use short task requests
- One task at a time works best
- Say which file or folder you want changed

- Keep a backup of your project folder
